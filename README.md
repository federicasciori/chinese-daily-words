# Chinese Daily Words

A simple, clean web app to learn 5 new Chinese words every day.

**Features:**
- 5 new words each day (automatically rotates)
- Pinyin (pronunciation) display
- Audio pronunciation (click the speaker icon)
- Everyday vocabulary for work and travel
- Works in any browser

## Live Demo

Visit the live site: [chinese-daily-words.vercel.app](https://chinese-daily-words.vercel.app)

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Vocabulary Categories

- Greetings & Courtesy
- Work & Business
- Time & Numbers
- Conversation
- Food & Dining
- Travel & Directions
- Useful Phrases

## How It Works

The app shows a different set of 5 words each day. Words are randomly selected from a library of 40+ everyday Chinese words, perfect for workplace communication and travel in China.

Click the speaker icon to hear each word pronounced in Mandarin Chinese.
